## 摘要
我们提出了一种生成具有多个离散化层的非一致三角形网格的框架。该框架利用了由前沿Delaunay算法生成的具有均匀单元尺寸的网格的特征结构性质。此类网格的大块区域表现出类似于规则三角形格子的结构化模式。由于这种结构，可以通过将单元分组为更大复合单元的连通子集来粗化大块区域。当重复应用时，该过程会产生具有多个离散化层的网格。对于复杂几何形状，该框架可用于创建复合多域网格，其中在每个子域内生成多个离散化层。我们展示了由所提框架生成的网格示例，并讨论其细化和粗化的后处理策略。生成的网格非常适用于自适应网格细化技术的应用。所提出的框架可以很容易地与支持非一致三角形网格的现有网格生成器和有限元求解器集成。

## Abstract
We present a framework for generating nonconforming triangular meshes with multiple discretization layers. The framework exploits characteristic structural properties of meshes produced by a frontal Delaunay algorithm with uniform element size. The bulk region of such meshes exhibits a structured pattern resembling a regular triangular lattice. Owing to this structure, the bulk region can be coarsened by grouping elements into connected subsets of larger composite elements. When applied repeatedly, this procedure produces a mesh with multiple discretization layers. For complex geometries, the framework can be used to create composite multidomain meshes, where multiple discretization layers are generated within each subdomain. We present examples of meshes generated by the proposed framework and discuss postprocessing strategies for their refinement and coarsening. The resulting meshes are well suited for the application of adaptive mesh refinement techniques. The proposed framework can be readily integrated with existing mesh generators and finite-element solvers that support nonconforming triangular meshes.

### 2.1 核心思想
- **统一的网格表示**：将非一致网格视为一致网格的极限情况，非一致界面用零面积三角形（void element）表示，其中悬挂节点位于共享边的中点。这样在组合层面网格始终保持一致，便于操作。
- **超三角剖分（Supertriangulation）**：定义一种数据结构，将原始网格中的一组三角形（核心三角形及其三个邻居）组合成一个“超三角形”。超三角形由原始网格中对应三角形的对边顶点构成。整个原始网格的这类超三角形集合称为“背景超三角剖分”。
- **迭代粗化**：利用超三角剖分的结构性质，通过过滤（平滑、剥离、清洁）和约简操作，提取出连接且紧凑的超三角形子集，将其作为粗化后的“内部网格”，其余部分作为“外部网格”。在内外网格接触处插入void element（将悬挂节点移动到对应中点），实现非一致连接。重复该过程（将新网格作为输入）即可生成多个离散化层。

### 2.2 关键技术细节
- **数据结构和运算符**：
  - `T_core(T)`：提取所有边都被相邻三角形共享的三角形（核心三角形）。
  - `T_sup(T_core)`：从核心三角形构造超三角形。
  - 过滤运算符：`F_clean`（剥离+迭代平滑）、`F_detach`（移除边界超三角形）、`F_reduce`（基于连通图从种子点约简）、`F_reduce_iter`（迭代约简直至获得紧凑超三角剖分）。
- **粗化算法（F_coarsen）**：
  1. **输入处理**：构造背景超三角剖分，可选边界分离，应用迭代约简得到紧凑超三角剖分，可选多次清洁。
  2. **网格分解**：将原始网格分为内部网格（紧凑超三角剖分本身）和外部网格（去除内部覆盖的所有原始三角形）。检查内外边界路径是否满足`2:1`比例关系（内边界节点数 = 外边界节点数的一半），否则终止。
  3. **网格重建**：将内外网格合并，并在接触边界上每个对应位置插入void element（将中间节点移到中点），从而在组合层面恢复连通性，形成新的非一致网格。
- **算法参数**：
  - `α_detach`：是否从原始边界分离超三角剖分。
  - `c0`：种子点，用于选择初始超三角形。
  - `n_shrink`：约简后清洁步骤的次数，控制离散化层之间的间隔。
- **后处理**：基于void element构建细化前沿和粗化前沿，实现局部或正面（frontal）的细化和粗化操作，且两者可逆（当子三角形满足红细化条件时）。另外可移除缺陷（不满足红细化条件的超三角形）。
