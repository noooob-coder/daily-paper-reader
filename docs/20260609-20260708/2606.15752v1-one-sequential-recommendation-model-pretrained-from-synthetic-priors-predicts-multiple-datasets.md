## 摘要
现有的序列推荐模型依赖于特定数据集的训练，其中学习到的参数会拟合训练数据的物品目录和观察到的交互分布。这限制了对新领域的泛化能力，通常需要从头重新训练。在这项工作中，我们提出了SRPFN，一种用于序列推荐的先验数据拟合网络——在目标域中无需任何基于梯度的参数更新，通过单次前向传递即可预测下一个物品。SRPFN在从涵盖多样化物物转换模式的合成先验中采样的2560万条序列上进行离线预训练，学习生成后验预测的下一个物品分布。在推理时，SRPFN通过以目标域的物物转换示例的支持集为条件来生成推荐，无需重新训练即可适应特定领域的模式。在五个基准测试和十个基线上的大量实验表明，SRPFN在几乎所有指标和数据集上达到最佳或次佳性能，同时计算效率远高于训练过的基线。这些结果证明，一个在合成先验上预训练的单一模型能够泛化到多样化的真实世界领域，为无需更新的序列推荐提供了一个框架。

## Abstract
Existing sequential recommendation models rely on dataset-specific training, where the learned parameters are fitted to the item catalog and the observed interaction distribution of the training data. This limits generalization to new domains, typically requiring retraining from scratch. In this work, we propose SRPFN, a Prior-data Fitted Network for sequential recommendation -- predicting the next item in a single forward pass without any gradient-based parameter updates in the target domain. SRPFN is pretrained offline on 25.6M sequences sampled from a synthetic prior that spans diverse item-to-item transition patterns, learning to produce posterior predictive next-item distributions. At inference time, SRPFN generates recommendations by conditioning on a support set of item-item transition examples from the target domain, adapting to domain-specific patterns without retraining. Extensive experiments on five benchmarks across 10 baselines show that SRPFN achieves the best or second-best performance across nearly all metrics and datasets, while being substantially more computationally efficient than trained baselines. These results establish that a single model pretrained on synthetic priors can generalize across diverse real-world domains, offering a framework for update-free sequential recommendation.

- **核心问题**：现有序列推荐模型（如GRU4Rec、SASRec）需针对每个数据集从头训练，将参数拟合到特定物品目录和交互分布，导致对新领域泛化性差。一旦切换到新数据集，必须重新训练，计算成本高且缺乏灵活性。
- **研究动机**：希望设计一种**无需目标域梯度更新**的序列推荐模型（称为“update-free inference”），仅通过一次前向传播即可适配新数据集，同时保持甚至超越传统需训练模型的性能。
- **背景**：作者观察到SASRec学习到的物品嵌入几何与低秩分解的转移统计量高度对齐（CKA值0.55~0.65），说明简单谱摘要即可反映过渡模式，这为使用低秩表示作为输入而不需梯度训练提供了理论基础。

## 2. 方法论

- **核心思想**：基于Prior-data Fitted Networks (PFNs) 范式，将序列推荐重新形式化为**条件推断问题**。模型在合成先验（synthetic prior）上预训练，学习近似**后验预测分布（PPD）**；推理时通过**支持集（support set）**（目标域的物物转换示例）进行条件化，在单次前向传播中输出下一物品的分布，无需任何梯度更新。
- **关键技术细节**：
  - **合成先验生成**：
    1. **物品图生成**：使用层次化度修正随机块模型 (hDCSBM) 生成具有层次社区结构、幂律度分布和核心-外围组织的物品图，得到随机转移核 **G**。
    2. **序列生成**：对每个用户，从幂律分布采样序列长度，从**几何分布**采样滞后ℓ（模拟短期依赖），然后基于参考状态（前ℓ个交互）从G的对应行采样下一物品，生成25.6M条合成序列。
  - **低秩输入表示**：
    - 从目标域训练数据构建**用户-物品交互矩阵 R** 和**物品-物品转移矩阵 P**（含多跳逆距离加权），经PPMI变换后，通过**截断随机SVD**分解，得到用户协同嵌入 **u_cf**、物品协同嵌入 **i_cf**、源端转移嵌入 **i_src**、目标端转移嵌入 **i_dst**。
  - **支持集构建**：对于每个测试用户，以其历史最后物品为源，从转移矩阵P中按概率采样 **|D_support|** 个目标物品，形成支持集。
  - **模型架构**：
    - **编码器**：对连续交互对 (i_{t-1}, i_t)，使用MLP生成上下文向量，结合用户嵌入形成状态，通过门控残差更新（gate基于余弦相似度特征）输出序列表示。
    - **Transformer + 交叉注意力**：因果Transformer编码历史序列得最后隐藏状态h，再对支持集内存进行多头交叉注意力，生成上下文向量c，通过门控融合得到最终后验表示 **ŷ**。
    - **评分函数**：ŷ与候选物品的编码向量（通过同一编码器计算）的内积。
  - **训练目标**：在合成数据上优化**softmax交叉熵损失**（负采样），该损失在最优情况下等价于贝叶斯PPD。
- **算法流程（文字说明）**：
  - **预训练阶段**：从合成先验采样转移核G → 生成随机游走序列 → 构造支持集 → 输入Transformer，优化交叉熵损失。
  - **推理阶段**：对目标域训练数据，计算低秩嵌入 → 对每个测试用户构建支持集 → 编码历史序列 → 交叉注意力融合支持信息 → 输出后验表示 → 与候选物品内积排序。

## 6. 主要结论与发现

- **性能领先**：SRPFN在5个数据集上平均比第二强基线提升 **7.53%**，在Beauty、Sports、Toys、Yelp的所有指标上排名第一，在LastFM上HR@1和HR@10也最高。
- **免更新高效**：无需目标域梯度更新，推理时间约1分钟/数据集，远低于需训练的基线（几分钟到十几小时）。
- **跨域泛化**：在4个未见Amazon类别上，SRPFN的HR@1远超所有免训练基线和预训练模型，证明了合成先验+条件推断的泛化能力。
- **流行度偏差缓解**：在Tail物品上SRPFN的HR@5提升幅度（49%~81%）显著高于Head物品（2.7%~10%），说明模型避免了过度偏向热门物品。
- **支持集的重要性**：|D_support|=1即可大幅提升性能（如Beauty上HR@1从0.1185升至0.2247），随大小增加持续改善。
- **架构设计有效**：源/目标非对称表示和门控残差更新均对性能有正向贡献。

## 7. 优点

- **创新性强**：首次将PFN思想引入序列推荐，提出**合成先验+条件推断**的免更新范式，从根本上解耦了模型与数据集的绑定。
- **计算效率革命性**：一次性预训练后，新增数据集仅需1分钟推理，相比传统训练节省大量计算资源，适合快速落地。
- **泛化能力突出**：不仅在同分布测试集上领先，还能直接泛化到未见类别，且对长尾物品友好。
- **设计合理**：低秩输入表示避免了ID嵌入学习；门控机制结合交叉注意力，平衡了历史序列和外部过渡证据。
- **实验充分**：涵盖多维度消融和跨域评估，结论可靠。

## 8. 不足与局限

- **合成先验的局限性**：当前先验基于静态图随机游走和几何滞后，未模拟**时间衰减、会话边界、重复消费**等真实时间动态。若能加入更丰富的时序结构，可能进一步提升性能。
- **部分场景仍有差距**：在LastFM（长序列、小物品）上，SRPFN的HR@5/NDCG@5略低于SASRec，说明需训练模型在特定设定下仍有优势。
- **全目录排名评估不足**：仅附录中报告了全目录结果，且SRPFN在NDCG@10上不如FEARec等，表明在无候选集过滤的全排名中优势可能减弱。
- **跨域泛化评估范围有限**：仅测试了4个Amazon类别，且未覆盖工业级大规模场景或不同交互模式（如社交、新闻）。
- **可解释性未讨论**：模型的后验预测机制黑盒，难以解释为何支持集条件化能适配新领域。
- **依赖低秩SVD**：SVD分解的计算和存储可能在大规模物品集上成为瓶颈，文中未讨论扩展性。
