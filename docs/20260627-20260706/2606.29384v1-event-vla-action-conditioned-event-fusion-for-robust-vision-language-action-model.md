## 摘要
视觉-语言-动作（VLA）模型已成为具身智能的重要范式。然而，现有VLA模型通常假设光照良好且稳定的室内环境，而真实世界的具身操作可能涉及光照变化导致的降质RGB观测，这给鲁棒的机器人操作带来了关键挑战。为弥补这一不足，我们提出Event-VLA，一种事件增强的VLA框架，用于在变化光照条件下实现可泛化的操作。我们将降质可见度下的VLA操作制定为以RGB为中心的策略的实用性鲁棒性问题，并引入事件流作为一种光照鲁棒、运动敏感的互补观测，以提升不同可见度水平下的鲁棒性。具体而言，不同于直接将事件特征合并到全局语义令牌空间中的传统多模态融合方法，Event-VLA通过动作查询路由路径注入事件信息。它使用可学习的动作查询从VLA推理过程中提取任务相关语义，并通过门控交叉注意力有选择地聚合事件令牌，以构建事件感知的动作表示。这种设计保留了预训练的RGB-语言语义先验，同时有效利用事件信息进行鲁棒的动作预测。仿真和真实世界部署实验表明，Event-VLA在正常光照下保持强大的操作性能，并在低光退化及近黑暗的真实环境中提高了成功率。

## Abstract
Vision-Language-Action (VLA) models have become an important paradigm of embodied AI. However, existing VLA models typically assume well-lit and stable indoor settings, while real-world embodied manipulation may involve degraded RGB observations caused by illumination shifts, posing critical challenges for robust robotic manipulation. To address this gap, we propose \textbf{Event-VLA}, an event-enhanced VLA framework for generalizable manipulation across varying illumination conditions. We formulate VLA-based manipulation under degraded visibility as a practical robustness problem for RGB-centric policies, and introduce event streams as an illumination-robust, motion-sensitive complementary observation to improve robustness across visibility levels. Specifically, unlike conventional multimodal fusion that directly merges event features into the global semantic token space, Event-VLA injects event information through an action-query routing pathway. It uses learnable action queries to extract task-relevant semantics from the VLA reasoning process, and selectively aggregates event tokens via gated cross-attention to construct event-aware action representations. This design preserves the pretrained RGB-language semantic priors while effectively leveraging event information for robust action prediction. Experiments in simulation and real-world deployment show that Event-VLA maintains strong manipulation performance under normal lighting and improves success rates under low-light degradation and near-dark real-world settings.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有视觉-语言-动作（VLA）模型通常假设稳定、光照良好的室内环境，但真实世界机器人操作常面临光照变化（如低光、近黑暗），导致RGB观测退化、语义接地不准确、动作预测不稳定。
- **动机**：事件相机具有高动态范围、低延迟、对运动和光照变化敏感的特性，可作为光照鲁棒、运动敏感的互补观测。但如何将异步高频事件流有效融入预训练的VLA模型（“事件到VLA接口问题”）是当前未充分研究的挑战。
- **贡献**：提出Event-VLA框架，通过动作条件的事件融合，在保持正常光照性能的同时显著提升低光鲁棒性。

## 2. 方法论：核心思想与关键技术
- **核心思想**：语义上下文由预训练VLA建模，事件信息通过受控方式注入动作路径，而非直接合并到全局语义令牌空间。
- **关键技术**：
  1. **PREI**：将异步事件流压缩为三通道物理残差图（瞬时、显著、持久），捕捉动作时间物理变化。
  2. **事件编码器**：通过特征蒸馏（从冻结的视觉教师）对齐事件特征与VLA隐藏空间，生成事件令牌（全局+空间令牌）。
  3. **动作条件接口**：VLM骨干输出上下文令牌后，通过**门控交叉注意力**将事件令牌与VLA隐藏令牌融合，再通过**查询引导路由**（通用查询、动作查询、事件查询）分离动作相关和事件相关令牌。动作头预测动作块；事件头预测未来PREI图（辅助损失）。
  4. **训练目标**：主损失为动作预测L1损失，辅助损失包括未来PREI重建损失和导数一致性损失，作为正则项。
- **设计原则**：事件令牌不参与VLM骨干的自注意力，仅通过后骨干融合影响动作输出，保护预训练语义先验。

## 6. 主要结论与发现
- **在正常光照下**：Event-VLA保持与强基线相当的SR（96.5% vs 96.3% MM-ACT），且关闭事件路径后性能几乎不变（96.2%），表明接口不干扰预训练路径。
- **在低光退化下**：RGB策略SR大幅下降（如LL-Severe下MM-ACT仅69.6%），而Event-VLA仍维持95.6%，证明事件残差提供了关键物理线索。
- **消融关键结论**：
  - PREI表示优于时间表面（95.6% vs 91.2%）。
  - 查询路由接口在SR和延迟间取得最佳平衡（95.6%，+2.157ms）。
  - 完整查询（通用+动作+事件）和辅助正则化带来一致增益。
- **真实世界**：Event-VLA在近黑暗条件下达52.5% SR，而π0仅15%、OpenVLA-OFT 12.5%，验证了实际有效性。

## 7. 优点
- **方法创新**：将事件信息注入动作路径而非全局语义空间，保护VLM预训练能力，同时高效利用事件信息。动作查询路由和门控交叉注意力设计巧妙。
- **表示设计**：PREI将事件流分解为瞬时/显著/持久三通道残差，兼顾时间分辨率和结构信息，比传统时间表面更稳定。
- **训练策略**：使用特征蒸馏对齐事件和RGB特征空间；辅助的未来PREI预测损失作为正则项，提升表示质量，且推理时可丢弃。
- **实验全面**：在多个仿真退化级别和真实场景验证，消融涵盖表示、接口、查询、正则化，并报告延迟，分析公平。
- **低延迟**：事件处理仅增加约2ms推理时间，适合实时操作。

## 8. 不足与局限
- **模拟器依赖**：LIBERO-Cross使用RGB-to-event模拟器生成事件，可能无法完全模拟真实事件相机的噪声、校准误差和触发动力学，存在模拟与现实差距。
- **真实世界验证有限**：仅4个任务，光照条件少，任务多样性不足；缺乏长时域、复杂交互场景的测试。
- **硬件要求**：需要额外的事件相机和同步系统，增加部署复杂度和成本。
- **适用范围窄**：主要针对光照退化（低光、近黑暗），未验证对其他退化（如运动模糊、遮挡）的鲁棒性。
- **实验统计**：真实世界每个条件仅10次实验，统计稳定性较低；仿真实验虽多但缺乏跨随机种子报告方差。
- **未讨论失败模式**：当目标物体事件响应弱（如低对比度、静止）时，事件信息可能无效（附录图8示例）。
