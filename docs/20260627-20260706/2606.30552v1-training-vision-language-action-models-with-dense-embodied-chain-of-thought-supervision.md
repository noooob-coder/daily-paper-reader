## 摘要
跨实体迁移在视觉-语言-动作（VLA）模型中仍然具有挑战性，因为不同机器人平台的低层状态和动作空间存在根本差异。我们观察到，操作背后的高层认知过程（包括场景感知、物体识别、任务规划和子任务分解）在各个实体之间在很大程度上是共享的。基于这一观察，我们提出了ZR-0，一个26亿参数的端到端VLA模型，该模型使用密集的具身思维链（ECoT）监督来对齐视觉-语言模型（VLM）中的跨实体表示。ZR-0采用双流架构：预训练的VLM（系统2）在训练期间生成结构化的ECoT推理，而基于扩散Transformer的动作专家（系统1）通过流匹配生成连续的动作块。这两个组件通过交叉注意力耦合，并采用注意力掩码限制动作专家仅使用输入提示特征，从而在推理时完全跳过ECoT生成而不损失任何性能。ZR-0在ProcCorpus-60M上进行预训练，该大规模数据集包含约6000万帧（约1000小时），来自超过40万条轨迹，密集的ECoT注释覆盖了96.8%的帧。我们在三个模拟基准上评估ZR-0，涵盖单臂（LIBERO）、双臂（RoboTwin 2.0）和类人（RoboCasa GR-1 Tabletop）实体，以及在xArm平台上的真实世界实验，展示了在所有设置中的强劲性能。代码和模型检查点可在https://github.com/RUCKBReasoning/ZR-0获取。

## Abstract
Cross-embodiment transfer in vision-language-action (VLA) models remains challenging because low-level state and action spaces differ fundamentally across robot platforms. We observe that the high-level cognitive process underlying manipulation, including scene perception, object identification, task planning, and sub-task decomposition, is largely shared across embodiments. Based on this observation, we present ZR-0, a 2.6 billion parameter end-to-end VLA model that uses dense Embodied Chain-of-Thought (ECoT) supervision to align cross-embodiment representations within the vision-language model (VLM). ZR-0 adopts a dual-stream architecture: a pre-trained VLM (System 2) generates structured ECoT reasoning during training, while a Diffusion Transformer-based action expert (System 1) produces continuous action chunks via flow matching. The two components are coupled through cross-attention, with an attention mask that restricts the action expert to input prompt features only, enabling ECoT generation to be entirely skipped at inference without any performance loss. ZR-0 is pre-trained on ProcCorpus-60M, a large-scale dataset comprising approximately 60 million frames (approximately 1,000 hours) from over 400K trajectories, with dense ECoT annotations covering 96.8% of all frames. We evaluate ZR-0 on three simulation benchmarks spanning single-arm (LIBERO), bimanual (RoboTwin 2.0), and humanoid (RoboCasa GR-1 Tabletop) embodiments, as well as real-world experiments on the xArm platform, demonstrating strong performance across all settings. Code and model checkpoints are available at https://github.com/RUCKBReasoning/ZR-0.

### 论文核心总结：通过密集具身思维链监督训练视觉-语言-动作模型

#### 1. 论文的核心问题与整体含义

-   **研究动机**：视觉-语言-动作模型的核心挑战在于**跨实体迁移（Cross-Embodiment Transfer）**。不同机器人平台（如不同自由度、控制接口的手臂）的低层状态和动作空间存在本质差异，现有的零填充、归一化等方法只能解决格式统一问题，无法实现**语义对齐**，即模型学到的仍是实体特定的特征，而非可迁移的共享知识。
-   **核心假设与观察**：作者观察到，尽管低层控制信号不同，但机器人执行操作任务的高层认知过程（如场景感知、物体识别、任务规划、子任务分解）在不同实体之间是高度共享的。这种“实体无关”的推理构成了可迁移的知识。
-   **整体含义**：本文提出一种方法，通过强制模型在训练过程中学习、对齐并利用这些共享的高层认知过程，来弥合不同实体之间的鸿沟，最终实现一个能有效控制多种不同形态机器人的通用基础模型。

#### 2. 论文提出的方法论：ZR-0

-   **核心思想**：利用**密集的具身思维链（Embodied Chain-of-Thought, ECoT）** 作为训练监督信号，驱动视觉-语言模型学习语义对齐、可迁移的表征。该ECoT将高层语言指令逐步分解为实体无关的规划、子目标和物体定位，从而实现对跨实体表征的对齐。
-   **关键技术细节**：
    1.  **双流架构（System 1 / System 2 框架）**：
        -   **System 2 (VLM)**：一个预训练的视觉-语言模型（基于Qwen3-VL-2B-Instruct），负责处理视觉观察和任务指令，并生成结构化的ECoT推理序列。其最后一层的隐藏状态作为特征传递给System 1。
        -   **System 1 (Action Expert)**：一个基于扩散Transformer的动作专家，通过流匹配（Flow Matching）直接生成连续的、高频的动作块。
    2.  **耦合与推理加速机制**：两个系统通过**交叉注意力（Cross-Attention）** 连接。关键创新在于使用一个**注意力掩码（Attention Mask）**，限制Action Expert只能关注VLM中输入提示（指令和图像）的特征，**忽略ECoT生成的文本token**。这确保了在推理时，只需对VLM进行一次前向传播（跳过昂贵的自回归ECoT生成），即可获得Action Expert所需的所有特征，无性能损失。
    3.  **密集ECoT监督**：每个帧的ECoT注释包含六个组成部分：
        -   场景描述：训练物体识别能力。
        -   进度评估：训练任务进展感知。
        -   未来计划：训练长时程规划。
        -   **待办动作**：将计划分解为“动词+物体”的原子性子任务（实体无关格式），是跨实体对齐的关键。
        -   **目标物体**：提供边界框，训练视觉定位。
        -   离散动作：作为连接到连续动作的桥梁。
-   **训练与推理流程**：
    -   **训练**：联合优化两个损失函数：`L_ntp`（ECoT的下一个词预测）和 `L_fm`（连续动作块的流匹配）。
    -   **推理**：输入指令、图像和机器人状态，VLM只编码输入提示，Action Expert从噪声开始，通过前向欧拉积分（流匹配）迭代N步去噪，生成动作块。整个过程约需90ms。
